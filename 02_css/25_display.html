블록레벨 요소 vs 인라인 요소

블록레벨 요소

  • display:block; (단, li는 display:list-item;)
  • 줄바꿈o: 위에서 아래로 배치(세로 배치)
  • html, body, div, h1~h6, p, ul, li, ...
  • width:100%; height:콘텐츠; border:none;
  • width, height, margin, padding, border를 지정할 수 있다.

인라인 요소

  • display:inline;
  • 줄바꿈x: 왼쪽에서 오른쪽으로 배치(가로 배치)
  • span, img, a, em, b, i, label, ...
  • width:콘텐츠; height:콘텐츠; border:none;
  • width, height는 지정되지 않는다.(단, 이미지는 예외)
  • margin은 좌우만 지정된다.
  • padding, border는 지정된다.
스팬 임시링크 강조 굵게 이텔릭

인라인 요소

  • display:inline-block;
  • 줄바꿈x: 왼쪽에서 오른쪽으로 배치(가로 배치)
  • button, input, ...
  • width, height, margin, padding, border를 지정할 수 있다.
  • button은 box-sizing:border-box; 이다.
